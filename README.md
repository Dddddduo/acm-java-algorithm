## 🤝 共建指南
1. **Issue规范**：提交问题请注明平台（如CF1923D）和错误描述
2. **PR准则**：
    - 新增算法模板需包含时空复杂度分析
    - 竞赛代码需通过对应平台测试
3. 推荐贡献方向：
    - 补充Codeforces 1400+题解
    - 完善动态规划状态压缩模板
